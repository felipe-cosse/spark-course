# Exercises: Course Home and Learning Strategy

Source: [PySpark course home](../README.md)

Estimated time: 35–50 minutes. Difficulty: introductory.

## Key terms reinforced

| Term | Definition |
|---|---|
| Learning objective | A specific capability a learner should demonstrate after instruction. |
| Dependency map | A diagram showing which concepts must be understood before others. |
| Evidence | An observable artifact proving a capability. |
| Learning contract | A personal agreement about schedule, outputs, review, and completion. |
| Reflection | Deliberate analysis of what was learned, misunderstood, or changed. |

## Exercise 1: Map the course

Choose one target role: data engineer, analytics engineer, ML engineer, or platform engineer. Create a dependency map with at least 12 course modules. Draw arrows where one module supplies concepts required by another.

### Deliverable

A diagram plus five sentences explaining the most important dependency chain.

### Hint

Start with distributed execution and schemas. Ask which later modules rely on partitions, grain, time, and idempotency.

## Exercise 2: Convert objectives into evidence

Select six outcomes from “What you will be able to do.” For each, define an artifact that would convince a skeptical reviewer you can perform it.

Do not use “I read the lesson” or “the code ran” as evidence.

### Deliverable

A table with outcome, artifact, verification method, and likely reviewer.

## Exercise 3: Write your learning contract

Create a four-week or fourteen-week schedule containing study blocks, exercise deadlines, review partners, and recovery time for missed work. Include a rule for when you may consult hints or solutions from other sources.

## Self-check

- Does the path preserve prerequisites?
- Does every selected objective have reproducible evidence?
- Is the schedule realistic about coding and review time?
- Does the contract include reflection and revision, not only first attempts?

## Stretch task

Design a short diagnostic assessment that places an experienced learner into the correct starting module without relying on self-reported confidence.
