# Exercises: Course Plan and Learning Paths

Source: [Course plan and learning paths](../../00-getting-started/01-course-plan.md)

Estimated time: 45–60 minutes. Difficulty: introductory.

## Key terms reinforced

| Term | Definition |
|---|---|
| Prerequisite | Knowledge required before attempting a later task. |
| Work product | A concrete artifact created through applied work. |
| Formative assessment | Low-stakes assessment used to guide learning while it is happening. |
| Summative assessment | Assessment used to judge achievement after a learning sequence. |
| Acceptance criteria | Observable conditions required for completion. |

## Exercise 1: Diagnose readiness

Rate your current evidence—not confidence—for Python, SQL, Linux, testing, and distributed-systems concepts. For each area, cite one artifact or mark it “not yet demonstrated.”

### Deliverable

A readiness table and a list of no more than three prerequisite activities.

## Exercise 2: Adapt the 14-week plan

Create a plan for one of these constraints:

- 3 hours per week for 20 weeks;
- 10 hours per week for 8 weeks;
- a five-person team with different roles;
- an experienced SQL analyst new to Python and distributed systems.

Preserve required concept dependencies and attach one assessed work product to every phase.

### Hint

Do not merely compress two weeks into one. Reduce scope, pair related work, and state what becomes optional.

## Exercise 3: Improve an acceptance criterion

Rewrite each weak criterion:

1. “The pipeline works.”
2. “The job is fast.”
3. “Bad rows are handled.”
4. “The stream is exactly once.”

Make each observable, scoped, and verifiable.

## Self-check

- Does the plan produce evidence rather than reading completion?
- Are assessments aligned with role goals?
- Are criteria measurable without relying on opinion?
- Is the workload achievable inside the stated time budget?

## Stretch task

Create a peer-review rotation that prevents one learner from being both sole implementer and sole reviewer of the same capstone dimension.
